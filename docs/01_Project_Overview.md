# ShopSense

## Project Description

ShopSense is a Multi-Vendor E-Commerce Analytics Platform that enables customers to shop from multiple vendors through a single marketplace while providing vendors and administrators with analytical insights into sales, inventory, and customer behavior.

## Stakeholders

- Customer
- Vendor
- Admin

## Core Modules

1. Authentication
2. Vendor Management
3. Product Management
4. Inventory Management
5. Order Management
6. Sales Reporting
7. Analytics (Future)
