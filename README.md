# ShopSense – Multi-Vendor E-Commerce Analytics Platform

## Overview

ShopSense is a Multi-Vendor E-Commerce Analytics Platform developed as part of the Infosys Customer Insights Platform Internship.

The platform enables:

- Customer shopping experience
- Vendor product management
- Marketplace administration
- Sales analytics
- Inventory monitoring
- Customer behaviour analysis
- Business intelligence dashboards

## Technology Stack

### Frontend
- React
- Tailwind CSS

### Backend
- FastAPI
- SQLAlchemy

### Database
- PostgreSQL

### Machine Learning
- Scikit-learn
- Pandas
- MLflow
